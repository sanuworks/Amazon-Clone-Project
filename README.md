# 🛒 Amazon Clone

A pixel-perfect front-end clone of Amazon's homepage built using pure HTML and CSS. Replicates the real Amazon UI including the navbar, search bar, product grid, footer, and more.

---

## 🖥️ Live Preview

> Open `index.html` directly in your browser to view the project.

---

## ✨ Features

- 🔝 **Navbar** — Logo, delivery address, search bar with category dropdown, language selector, sign-in, orders, and cart
- 🔍 **Search Bar** — Category dropdown with 25+ departments and expandable search input
- 📋 **Navigation Panel** — Secondary menu with Today's Deals, Prime Video, Registry, Gift Cards, and more
- 🖼️ **Hero Section** — Full-width banner image with India redirect message
- 🛍️ **Product Grid** — 8 product cards with images and "Shop Now" links
- 🏆 **Best Sellers Section** — Clothing, Shoes & Jewelry category with 8 product images
- 👤 **Personalized Section** — Sign-in prompt with "New customers? Start here" link
- 🔗 **Footer** — 4-column info section with Back to Top button and copyright

## ⚙️ Setup & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/sanuworks/amazon-clone.git
cd amazon-clone
```

### 2. Open in Browser
Just double-click `index.html` or open it with Live Server in VS Code.

> No installations or dependencies required. Pure HTML & CSS project.
